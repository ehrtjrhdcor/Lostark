시작하기

이 웹페이지를 실행하려면 다음 단계를 따라주세요:

  1. npm install 명령어로 필요한 의존성을 설치하세요
  2. npm start 명령어로 서버를 시작하세요
  3. 브라우저에서 http://localhost:1707를 방문하세요
